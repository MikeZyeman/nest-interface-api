# Nest-Interface-API

## Description

An API for an Shopify Interface Application. This is an example for a Saas Project, which transports product datas between Shopify and the api itself via the interface application.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
